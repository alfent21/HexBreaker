# HexBreaker 操作ガイド

## エディター操作

### マウス操作

| 操作 | 動作 |
|------|------|
| **左クリック** | ツール使用（ブラシ配置、ライン描画など） |
| **左ドラッグ** | 連続配置/消去、選択範囲、頂点移動 |
| **右クリック** | ツール切替 / ブロックメニュー / ライン頂点削除 |
| **ミドルボタン + ドラッグ** | キャンバスをパン（移動） |
| **ホイール回転** | ズームイン/アウト（カーソル位置中心） |
| **ダブルクリック** | ライン描画完了 / 選択ライン上に頂点追加 |

### キーボードショートカット

#### ツール切替

| キー | ツール |
|------|--------|
| `B` | ブラシ |
| `E` | 消しゴム |
| `L` | ラインツール |
| `G` | 塗りつぶし |
| `I` | スポイト |
| `V` | 選択ツール |

#### 修飾キー

| キー | 動作 |
|------|------|
| `Space + ドラッグ` | キャンバスをパン |
| `Shift + ブラシ` | 消しゴムモード |
| `Shift + 選択` | 選択に追加 |

#### 編集操作

| キー | 動作 |
|------|------|
| `1`〜`9` | 耐久度を設定 |
| `[` | ブラシサイズ縮小 |
| `]` | ブラシサイズ拡大 |
| `H` | グリッド表示/非表示 |
| `Delete` | 選択したライン/ブロックを削除 |
| `Enter` | 選択範囲を現在色で塗りつぶし |
| `Escape` | ライン描画キャンセル / 選択解除 |

#### ファイル操作

| キー | 動作 |
|------|------|
| `Ctrl+S` | プロジェクト保存 |
| `Ctrl+Z` | 元に戻す |
| `Ctrl+Y` / `Ctrl+Shift+Z` | やり直し |

### ラインツール専用

| 操作 | 動作 |
|------|------|
| 左クリック | 頂点を追加 |
| ダブルクリック | 描画完了 |
| 右クリック（描画中） | 直前の頂点を削除 |
| 右クリック（頂点上） | 頂点を削除 |
| 頂点ドラッグ | 頂点を移動（スナップ対応） |

## ゲーム操作

### 基本操作

| 操作 | 動作 |
|------|------|
| マウス移動 | パドル移動 |
| 左クリック | ボール発射 / ウエポン使用（MAGNET/GHOST時） |
| `Escape` | ポーズ/再開 |

### ウエポン購入

| キー | ウエポン | コスト | 効果 |
|------|----------|--------|------|
| `1` | SLOW | 1 | ボール速度×0.6（15秒） |
| `2` | EXPAND | 2 | パドル幅拡大（20秒） |
| `3` | DOUBLE | 2 | ボール複製 |
| `4` | LASER | 3 | レーザー発射（5発） |
| `5` | SHIELD | 4 | ミスライン防御 |
| `6` | MAGNET | 4 | ボール吸着（20秒） |
| `7` | GHOST | 4 | ブロック貫通（15秒） |

※ フッターのウエポンスロットをクリックでも購入可能
